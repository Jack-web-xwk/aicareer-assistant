# utils/

## 功能说明
通用工具函数模块，包含文件处理、格式转换、日志记录等辅助功能。

## 开发状态
- [ ] 待开发
- [x] 开发中
- [ ] 已完成

## 依赖关系
- Python 标准库
- typing（类型注解）

## 文件说明
| 文件 | 说明 |
|------|------|
| __init__.py | 模块初始化 |
| file_utils.py | 文件操作工具 |
| text_utils.py | 文本处理工具 |
| logger.py | 日志配置 |

## 注意事项
- 工具函数应保持纯函数特性
- 避免在工具模块中引入业务逻辑
- 所有函数需添加类型注解和文档字符串
