## 💡 实战经验

- **Claude Code 插件生态爆发**：Superpowers、Claude HUD、Claude Subconscious 三个项目展示了 Agent 插件化的三种范式——开发方法论增强、运行时可观测性、后台记忆增强。值得研究它们的架构设计。

- **DeerFlow 2.0 的 SuperAgent 架构**：沙箱 + 记忆 + 子代理 + 技能系统，是目前最完整的开源长周期 Agent 框架。适合需要 Agent 自主运行数小时的场景。

- **Supermemory 的 RAG+Memory 一体化方案**：解决了"AI 每次对话都忘记一切"的痛点，一个 API 搞定向量检索、记忆管理、用户画像、多模态处理。对 RAG 应用开发者极具参考价值。

- **everything-claude-code 的 Agent 工程实践**：10+ 个月的高强度日常使用沉淀，Token 优化、记忆持久化、连续学习、验证循环、并行化、子代理编排的最佳实践总结。
