# AI应用实战日报 - 2026年3月28日

## 🔥 本周要闻

**1. OpenAI 发布 Codex Plugin：支持一键打包 Skills 和 MCP 配置**
OpenAI 正式推出 Codex Plugin 服务，开发者可将 Skills、应用集成和 MCP Server 配置打包为可安装的插件，实现团队间一键同步开发配置。目前已集成 Slack、Figma、Notion、Cloudflare 等主流工具。Codex 月活开发者已超 100 万。
来源：https://www.aibase.com/news/26630

**2. Uber 借助 AI Agent + MCP 实现设计文档自动化，时间从数周缩短到几分钟**
Uber 工程团队发布 uSpec 系统，通过 Cursor IDE 中的 AI Agent 与本地 Figma Desktop 会话桥接，利用 Figma Console MCP 自动"爬取"组件树并生成跨平台技术规范，数据全程本地化。
来源：https://www.infoq.cn/article/ByY97HEFsSdPTMIVzJuP

**3. 清华大学提出动态化与参数化 RAG 新范式（DRAGIN / DecoupledRAG / Parametric RAG）**
DRAGIN 基于模型内部状态动态判断检索时机；DecoupledRAG 通过交叉注意力将检索文档与提示词解耦；Parametric RAG 将文档编码为 LoRA 权重直接注入 FFN 层，实现参数化知识注入。
来源：https://www.infoq.cn/article/bdHrWhvKqigAAd3ldh53

**4. 腾讯正式发布 Agent 产品全景图，构建 AI 时代全栈引擎**
腾讯系统性展示 Agent 产品架构：底层 AI Infra 智算基座 + 中层 TokenHub 大模型服务平台 + 上层 Skills 工具箱，深度集成微信、企微、QQ 等国民级入口。
来源：https://www.aibase.com/news/26628

**5. Anthropic 调整 Claude 用量限制机制，引入"峰谷错峰"模式**
太平洋时间 05:00-11:00 高峰时段，Claude 会话配额消耗加速，约 7% 的重度用户会受影响，但每周总配额不变。
来源：https://www.aibase.com/news/26622

## 💡 应用洞察

**MCP 正从概念走向工业落地。** Uber 的 uSpec 和腾讯的 Skillhub 表明 Model Context Protocol 正在被大厂采纳为企业级 Agent 的标准化工具接入层。

**RAG 技术正经历范式升级。** 清华团队的三项研究代表了 RAG 的三个进化方向：从静态检索走向动态按需检索、从提示词注入走向交叉注意力解耦、从上下文拼接走向参数化注入。
