## 🔥 要闻

### 1️⃣ TAPO：翻译增强策略优化，提升多语言数学推理
提出 TAPO (Translation Augmented Policy Optimization)，通过翻译增强的策略优化方法，利用 GRPO 框架提升 LLM 在多语言数学推理任务上的表现。该工作探索了 RL 训练在跨语言推理场景的边界。
🔗 [arXiv:2603 (3月26日)](https://arxiv.org/search/?query=TAPO+Translation+Augmented+Policy+Optimization&searchtype=all)

### 2️⃣ MoE-GRPO：用 RL 优化视觉语言模型中的混合专家架构
将 GRPO 扩展到 Vision-Language MoE 模型，通过强化学习优化稀疏激活的专家参数子集，在降低计算开销的同时维持模型能力。这是 RLHF/GRPO 在多模态 MoE 架构上的前沿探索。
🔗 [arXiv (3月25日)](https://arxiv.org/search/?query=MoE-GRPO+Mixture+Experts+Vision+Language&searchtype=all)

### 3️⃣ FedPDPO：联邦个性化 DPO 对齐
提出 FedPDPO (Federated Personalized DPO)，解决联邦学习场景下 DPO 对齐的核心挑战：去中心化、隐私敏感且高度非 IID 的偏好数据。在联邦分布式环境中实现个性化的 LLM 对齐。
🔗 [arXiv (3月20日)](https://arxiv.org/search/?query=FedPDPO+Federated+Personalized+DPO&searchtype=all)

### 4️⃣ Balanced DPO：通过平衡偏好对提升安全对齐
提出 Balanced DPO 方法，通过改进正负样本配比策略来增强 LLM 的安全对齐效果，解决传统 DPO 在安全场景下的偏差问题。
🔗 [arXiv (3月24日)](https://arxiv.org/search/?query=Balanced+DPO+Safety+Alignment&searchtype=all)

### 5️⃣ TuneShift-KD：微调模型的知识蒸馏与迁移
提出 TuneShift-KD 框架，针对微调后模型的知识蒸馏问题，实现将多个领域微调模型的知识高效迁移到基础模型中，减少多任务微调的计算成本。
🔗 [arXiv (3月25日)](https://arxiv.org/search/?query=TuneShift-KD+Knowledge+Distillation+Fine-tuned&searchtype=all)

## 🛠️ 工具与框架

### 🤗 PEFT 库 — 持续扩展中
PEFT 库近期新增多个微调方法支持：
- **RoAd (2D Rotary Adaptation)** — 仅使用逐元素乘法学习 2D 旋转矩阵，推理速度极快，支持混合 adapter 批次
- **ALoRA (Activated LoRA)** — 根据输入 token 序列选择性启用 LoRA adapter，优化 KV cache 复用
- **Arrow + GenKnowSub** — 多 LoRA 间的动态路由算法
- **WaveFT (Wavelet Fine-Tuning)** — 在残差矩阵的小波域进行稀疏更新，参数效率极高
- **DeLoRA (Decoupled LoRA)** — 解耦角度和幅度，类似 DoRA 但更好防止发散
- **OSF (Orthogonal Fine-Tuning)** — 冻结高秩子空间，投影梯度到低秩空间

🔗 [PEFT GitHub](https://github.com/huggingface/peft)

## 💡 今日洞察

本周微调领域最明显的趋势是 **RLVR (Reinforcement Learning with Verifiable Rewards) 的全面爆发**。从 DeepSeek R1 开始的 GRPO 范式，正在快速扩散到多模态（MoE-GRPO、VL Token Reweighting）、多语言（TAPO）和代码生成领域。与传统 RLHF 依赖人类标注不同，RLVR 用可验证的奖励信号（如数学正确性、代码运行结果）替代人工偏好，大幅降低了对齐成本。同时，DPO 方向也在持续演进——联邦个性化 DPO（FedPDPO）和安全性专项优化（Balanced DPO）表明社区正在从"通用对齐"走向"场景化对齐"。PEFT 库新增的 RoAd、DeLoRA、OSF 等方法则显示出参数高效微调仍在快速迭代，2D 旋转、小波域稀疏化等数学技巧正成为新热点。
